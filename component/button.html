Button 按钮 ​

常用的操作按钮。

基础用法 ​

使用 type、round 来定义按钮的样式。

<template>
  <p>
    <qi-button>Default</qi-button>
    <qi-button type="primary">Primary</qi-button>
    <qi-button type="success">Success</qi-button>
    <qi-button type="info">Info</qi-button>
    <qi-button type="warning">Warning</qi-button>
    <qi-button type="danger">Danger</qi-button>
  </p>
  <p>
    <qi-button round>Round</qi-button>
    <qi-button type="primary" round>Primary</qi-button>
    <qi-button type="success" round>Success</qi-button>
    <qi-button type="info" round>Info</qi-button>
    <qi-button type="warning" round>Warning</qi-button>
    <qi-button type="danger" round>Danger</qi-button>
  </p>
</template>

禁用状态 ​

你可以使用 disabled 属性来定义按钮是否被禁用。

使用 disabled 属性来控制按钮是否为禁用状态。 该属性接受一个 Boolean 类型的值。

<template>
  <p>
    <qi-button disabled>Default</qi-button>
    <qi-button type="primary" disabled>Primary</qi-button>
    <qi-button type="success" disabled>Success</qi-button>
    <qi-button type="info" disabled>Info</qi-button>
    <qi-button type="warning" disabled>Warning</qi-button>
    <qi-button type="danger" disabled>Danger</qi-button>
  </p>

  <p>
    <qi-button plain disabled>Plain</qi-button>
    <qi-button type="primary" plain disabled>Primary</qi-button>
    <qi-button type="success" plain disabled>Success</qi-button>
    <qi-button type="info" plain disabled>Info</qi-button>
    <qi-button type="warning" plain disabled>Warning</qi-button>
    <qi-button type="danger" plain disabled>Danger</qi-button>
  </p>
</template>

加载状态按钮 ​

点击按钮来加载数据，并向用户反馈加载状态。

通过设置 loading 属性为 true 来显示加载中状态。

<template>
  <p>
    <qi-button type="primary" loading>Primary</qi-button>
    <qi-button type="primary">Primary</qi-button>
  </p>
</template>

调整尺寸 ​

除了默认的大小，按钮组件还提供了几种额外的尺寸可供选择，以便适配不同的场景。

使用 size 属性额外配置尺寸，可使用 large 和 small 两种值。

<template>
  <p>
    <qi-button size="large">large</qi-button>
    <qi-button>medium</qi-button>
    <qi-button size="small">small</qi-button>
  </p>
</template>

自定义图标 ​

您可以使用 icon 插槽传入svg组件，自定义您的按钮图标。

icon-placement属性用于修改图标位置。

<template>
  <div>
    <qi-button size="large" icon="ghost"> 启林 </qi-button>
    <qi-button size="large" type="primary" icon="ghost" icon-placement="right">
      启林
    </qi-button>
  </div>
</template>

API ​

Attributes 属性 ​

属性名说明类型默认值
size尺寸enum : 'large' / 'medium' / 'small'—
type类型enum : 'primary' / 'success' / 'warning' / 'danger' / 'info' / 'default'—
loading是否显示加载中booleanfalse
round是否为圆角按钮booleanfalse
disabled是否禁用booleanfalse
native-type原生 type 属性enum : 'button' / 'submit' / 'reset''button'
icon-placement图标所在位置enum : 'left' / 'right''left'

事件名说明类型

mousedown鼠标按下事件Function : (event: MouseEvent) => <MouseEvent>event
