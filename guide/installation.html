使用包管理器 ​

我们建议您使用包管理器（如 npm、pnpm）安装 QiLin-UI，然后您就可以使用打包工具，例如 Vite 或 webpack。

npm
$ npm withInstall qilin-ui -D
pnpm
$ pnpm withInstall qilin-ui -D
